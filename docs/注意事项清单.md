# AI小芒 运行时注意事项清单

本文档涵盖开发、测试、发布各阶段的关键注意事项。

---

## 一、运行时问题排查

### 1.1 应用崩溃

| 现象 | 原因 | 解决方案 |
|------|------|---------|
| 启动即崩溃 | 摄像头/麦克风权限未声明 | 检查 `AndroidManifest.xml` 和 `Info.plist` |
| 启动即崩溃 | TFLite 模型文件损坏 | 重新下载，检查文件头非 HTML |
| 运行时崩溃 | 内存不足（VL 模型） | 设备 ≤4GB 内存时禁用 VL，见 `hardware_info.dart` |
| 运行时崩溃 | 3D 球体 GPU 不支持 | 自动降级 2D，参考 `ai_3d_ball.dart` |

### 1.2 模型加载失败

| 原因 | 排查方法 |
|------|---------|
| 文件路径错误 | 检查 `offline_ai_engine.dart` `_getModelPath()` 返回值 |
| TFLite 格式不匹配 | 确认使用 `.tflite` 文件，非 `.onnx` |
| GGUF mmap 不支持 | Android 10/11 禁用 mmap，见 `platform_utils.dart` |

### 1.3 录音问题

| 现象 | 原因 | 解决方案 |
|------|------|---------|
| 无声音录制 | 采样率非 16KHz | Whisper 必须 16KHz mono PCM，参考 `audio_recorder_service.dart` |
| 录音模糊 | PCM 数据被压缩/转码 | 确保无重采样，直接 PCM 原始数据 |
| iOS 录音失败 | 后台音频未配置 | `Info.plist` 添加 `UIBackgroundModes: audio` |

### 1.4 网络问题

| 现象 | 原因 | 解决方案 |
|------|------|---------|
| SSE 连接失败 | Nginx 未配置 `proxy_buffering off` | 参考 `docs/nginx_sse_config.md` |
| SSE 断开 | 弱网 5 次重连后自动切离线 | 正常行为，参考 `sse_stream_service.dart` |
| 云端无响应 | 阿里云凭证未配置 | 检查 `backend/.env` 各项 KEY 是否填入 |
| TTS 无声音 | 阿里云 TTS 额度超限 | 观察日志 `"quota_exceeded"`，切换离线模式 |

### 1.5 阿里云服务问题

| 服务 | 常见错误 | 排查方式 |
|------|---------|---------|
| VL | `DASHSCOPE_API_KEY not configured` | 检查 `backend/.env` |
| ASR | WebSocket 连接失败 | 检查 `ALIYUN_ASR_APP_KEY` 和 region |
| TTS | 合成返回 None | 检查 `ALIYUN_TTS_APP_KEY`，确认日额度 |
| 所有服务 | `InvalidAccessKeyId` | AccessKey 过期，重新创建 |
| 所有服务 | `Throttling` | 请求频率超限，降低客户端上传频率 |

## 二、合规检查清单

发布前必须确认以下各项：

### 2.1 隐私合规

- [ ] `AndroidManifest.xml` 已声明 `CAMERA`、`RECORD_AUDIO` 权限
- [ ] `Info.plist` 已声明 `NSCameraUsageDescription`、`NSMicrophoneUsageDescription`
- [ ] 权限申请已在运行时向用户说明用途（不要自动静默授权）
- [ ] 应用内未收集用户生物特征数据（声音/图像）上传到第三方服务器
- [ ] AI 回答内容中未包含未经授权的版权内容

### 2.2 平台合规

- [ ] iOS `Info.plist` 设置了 `UIBackgroundModes: audio`
- [ ] Android `AndroidManifest.xml` 声明了 `FOREGROUND_SERVICE` + `foregroundServiceType`
- [ ] 应用在应用商店审核时提供权限用途说明文案
- [ ] iOS 最低版本 ≥ 13（Metal 加速要求）
- [ ] Android 最低版本 ≥ 21（目标 Android 12+）

### 2.3 安全合规

- [ ] `backend/.env` 未提交到代码仓库（已在 `.gitignore` 中）
- [ ] 阿里云 AccessKey 使用子账号 AccessKey（非主账号）
- [ ] 阿里云 AccessKey 设置了 IP 白名单限制
- [ ] 生产环境 SSE 使用 HTTPS
- [ ] 生产环境 JWT token 校验已启用（当前开发模式为简版校验）

### 2.4 成本合规

- [ ] `DAILY_TTS_QUOTA` 已设置每日额度上限
- [ ] 欠费时自动切离线模式逻辑已验证
- [ ] 模型推理异常时不超过 30 秒自动超时
- [ ] `SESSION_TIMEOUT` 设置了会话过期（默认 600 秒）

## 三、已知限制

1. **离线模式实际不可用**：Whisper/VL 推理部分为骨架代码，需下载真实模型文件并集成 `tflite_flutter` / `llama_cpp_dart` 后才能工作
2. **阿里云凭证必须配置**：`DEBUG=false` 时所有服务需要真实阿里云凭证
3. **Android 10/11 GGUF mmap**：会自动降级到 CPU 模式，性能降低
4. **iOS Metal 加速**：仅 iOS 16+ 支持，iOS 15 自动降级 CPU
5. **长对话上下文**：超过 `MAX_CONTEXT_ROUNDS`（默认10轮）后自动截断最早对话
6. **TTS 分段限制**：单片段文本少于5字会被跳过
7. **球体 OBJ 文件**：当前 `assets/ball.obj` 未生成，如需 3D 球体需自行生成
8. **后端 ASR 集成**：当客户端未做 ASR 时，后端 `AliyunASR` 自动接管语音转文本（需配置 `ALIYUN_ASR_APP_KEY`）

## 四、开发调试技巧

### 4.1 启用 DEBUG 模式（无需阿里云凭证）

```bash
# backend/.env
DEBUG=true
```

### 4.2 查看后端日志

```bash
cd backend
uvicorn main:app --reload --port 8000 --log-level debug
```

### 4.3 检查模型是否加载

Flutter 端日志搜索：
```
[OfflineAIEngine] Whisper-tiny loaded successfully
[OfflineAIEngine] VL GGUF loaded from: ...
```

### 4.4 SSE 调试

```bash
# 手动测试 SSE
curl -N "http://localhost:8000/sse/chat?ctxId=test&token=dev_token"
```

### 4.5 弱网模拟

Android：
```bash
adb shell settings put global background_data 0
# 或使用 Network Emulator
```

iOS：
```bash
# Xcode → Device → Network Link Conditioner
```
