# AI小芒 — 注意事项清单

## 一、运行前准备

### 1.1 模型文件下载
```bash
cd assets/
chmod +x download_models.sh
export HF_TOKEN=your_huggingface_token  # 可选，无token部分模型下载受限
./download_models.sh
```

### 1.2 后端密钥配置
```bash
cd backend/
cp .env.example .env
# 编辑 .env，填入真实阿里云凭证
```

### 1.3 依赖安装
```bash
# Flutter 客户端
flutter pub get

# Python 后端
cd backend
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## 三、常见问题

### Q: 摄像头黑屏/权限被拒
A: 检查 `Info.plist` 中的 `NSCameraUsageDescription` 是否存在；检查 `AndroidManifest.xml` 中 `CAMERA` 权限是否声明；真机需要在设置中手动授权。

### Q: 麦克风录音无响应
A: 确认 `Info.plist` 中 `NSMicrophoneUsageDescription` 已声明；`UIBackgroundModes` 中包含 `audio`；麦克风采样率必须固定 16KHz。

### Q: 离线模式提示"模型未加载"
A: 检查 `assets/` 目录下是否存在 `whisper-tiny-int8.tflite` 和 `Qwen-VL-2B-Q4_K_M.gguf`；检查模型文件大小是否正常（拒绝下载不完整文件）。

### Q: SSE 连接断开
A: 检查后端是否运行在 `localhost:8000`；检查 Nginx 配置中 `proxy_buffering off` 是否正确；检查 `proxy_read_timeout 10m` 是否生效。

### Q: iOS 编译失败
A: 确保 Xcode 版本 ≥ 15.0；确保 `ios/Podfile` 中 `platform :ios, '16.0'`；运行 `cd ios && pod install --repo-update`。

### Q: Android 打包失败
A: 检查 `minSdk = 21`；检查 NDK 版本（建议 25.x）；删除 `android/.gradle` 缓存后重试。

### Q: 内存不足崩溃
A: 检查设备内存是否 ≤ 4GB；确认 `isLowMemoryDevice` 降级触发；离线模式下可手动关闭视觉模型以节省内存。

## 四、性能调优

### 抽帧优化
- 抽帧间隔固定 800ms，不得随意缩短
- 分辨率固定 640x480，JPG 质量 65
- 后端丢弃超速帧（间隔 < 500ms）

### 内存优化
- 端侧模型加载后不再释放（避免反复加载开销）
- 内存 > 3.5GB 时触发 GC 回收（`System.gc()`）
- 极低内存设备（≤2GB）强制禁用视觉模型

### 网络优化
- SSE 心跳 3s/次，超时 5s 重连，5 次上限后切离线
- 音频分片 200ms 一片，base64 编码后上传
- 弱网（移动数据）不自动重试，避免流量浪费

- [ ] SSE 心跳和超时配置正确

## 六、安全红线

1. **严禁**将 `accessKey`/`accessSecret` 硬编码到代码中
2. **严禁**将 `.env` 提交到 git
3. **严禁**将模型文件（`.tflite`/`.gguf`）提交到 git
4. **严禁**在离线模式下发送任何网络请求（含埋点）
5. **必须**使用 HTTPS 访问生产环境后端
