# AI小芒 — 注意事项清单

## 一、运行时问题排查

### 1.1 Flutter 端

| 现象 | 可能原因 | 解决方案 |
|------|----------|----------|
| 编译报错 `AnimatedBuilder` | Flutter 3.x 使用 `ListenableBuilder` | 已修复，替换为 `ListenableBuilder` |
| 摄像头权限弹窗不出现 | Info.plist 未配置 `NSCameraUsageDescription` | 已修复 |
| 麦克风权限弹窗不出现 | Info.plist 未配置 `NSMicrophoneUsageDescription` | 已修复 |
| 后台音频中断 | iOS 未配置 `UIBackgroundModes: audio` | 已修复 |
| 后台进程被杀死 | Android ForegroundService 未启动 | 已修复，MethodChannel 已桥接 |
| 录音数据上传失败 | 离线模式下 SSE 连接未建立 | 确保云端模式或切换离线 |
| AI 球体不显示 | iOS 暗黑模式下强制禁用 3D | 已修复，移除反向判断 |
| 语音播报后不回到 idle | `onSpeakComplete` 回调未正确注册 | 已修复，移至 `init()` |
| 消息不显示在 ChatPanel | 云端模式未调用 `addUserMessage` | 已修复 |
| 帧上传 HTTP Client 泄漏 | 每帧创建新 Client 未关闭 | 已修复，添加 `finally` 关闭 |

### 1.2 后端

| 现象 | 可能原因 | 解决方案 |
|------|----------|----------|
| SSE 连接失败 | Token 校验失败 | `backend/.env` 中配置 token，客户端 baseUrl 需匹配 |
| 推理返回 Mock 内容 | `DEBUG=true` 未关闭 | `.env` 中设置 `DEBUG=false` |
| TTS 无声音 | `DASHSCOPE_API_KEY` 未配置 | 在百炼平台获取 API Key |
| VL 返回错误 | 模型不支持视觉理解 | 确认使用的是 VL 模型（非 text-only） |
| 配额未生效 | TTS 配额估算不准确 | 已修复，优先使用服务器返回的真实音频时长 |
| WebSocket 超时 | 无超时控制，连接卡死 | 已修复，添加 `open_timeout=10, close_timeout=5` |

### 1.3 iOS 真机特定

| 现象 | 可能原因 | 解决方案 |
|------|----------|----------|
| Bundle Identifier 冲突 | `com.example.aiVideo` 被占用 | 修改 Xcode 中的 Bundle Identifier 为唯一值 |
| 无 provisioning profile | 开发团队未选择 | Xcode → Runner → Signing & Capabilities → Team |
| 摄像头黑屏 | 权限未授权 | 系统设置 → 隐私与安全性 → 摄像头 → 允许 AI小芒 |

### 1.4 Android 特定

| 现象 | 可能原因 | 解决方案 |
|------|----------|----------|
| 前台通知不显示 | Android 13+ 需要 `POST_NOTIFICATIONS` 权限 | 首次启动申请通知权限 |
| 摄像头预览黑屏 | CameraController 未初始化 | 检查 `camera` 插件版本兼容性 |
| 进程被杀死 | ForegroundService 通知渠道未创建 | 已修复，SDK >= 26 时自动创建渠道 |

## 二、合规检查清单

### 2.1 隐私合规

- [ ] 摄像头/麦克风权限说明文案清晰（已在 Info.plist 配置）
- [ ] 不在日志中打印用户语音/图像原始数据
- [ ] 端侧模型推理无需上传用户数据（离线模式）
- [ ] 云端模式下数据传输使用 HTTPS（确保生产环境 SSL）

### 2.2 平台规范

- [ ] iOS：遵守 App Store 审核指南（后台音频必须有实际用途）
- [ ] Android：遵守 Google Play 隐私政策（数据安全表单）
- [ ] 华为应用市场：如需上架需额外配置 HMS

### 2.3 阿里云服务合规

- [ ] API Key 仅存储在服务端 `.env`，不硬编码
- [ ] DashScope API 调用遵守使用限制
- [ ] TTS 音色版权由阿里云授权，无需额外付费

## 三、调试技巧

### 3.1 Flutter 调试

```bash
# 开启详细日志
flutter run --verbose

# 打印 AppState 状态变更
flutter logs | grep AppState

# 打印 AI 引擎日志
flutter logs | grep OfflineAIEngine

# 查看 SSE 连接状态
flutter logs | grep SseStreamService
```

### 3.2 后端调试

```bash
# 启动开发服务器（自动重载）
cd backend && uvicorn main:app --reload --port 8000

# 开启 DEBUG 模式（Mock 响应，无需阿里云凭证）
# 在 .env 中设置 DEBUG=true

# 查看会话清理日志
uvicorn main:app --reload --port 8000 --log-level debug
```

### 3.3 网络抓包

```bash
# 查看 Flutter 端请求
# Android: adb logcat | grep HTTP
# iOS: Xcode → Devices → Console

# 查看后端请求
# 使用 curl 测试 SSE 端点
curl -N "http://localhost:8000/sse/chat?ctxId=test&token=dev_token"
```
