# AI小芒 — 交付文档

## 一、项目概述

AI小芒 是一款 **Flutter + FastAPI** 的跨平台 AI 视觉语音对话助手，支持离线端侧推理和阿里云云端增强两种运行模式。

### 1.1 核心能力

| 能力 | 离线模式 | 云端模式 |
|------|----------|----------|
| 语音识别 | Whisper-tiny（端侧 TFLite） | 阿里云实时 ASR |
| 视觉理解 | Qwen2-VL-2B（端侧 GGUF） | 通义千问 VL |
| 语音播报 | FlutterTTS（本地） | 阿里云流式 TTS |
| 实时对话 | 本地流式响应 | SSE + 流式 TTS |

### 1.2 技术栈

| 层级 | 技术选型 |
|------|----------|
| 客户端 | Flutter 3.x, Dart 3.x |
| 状态管理 | Provider |
| 端侧 AI | tflite_flutter, llama_cpp_dart |
| 云端推理 | FastAPI, SSE, 阿里云 DashScope |
| 部署 | Docker, Nginx（可选） |

## 三、部署指南

### 3.1 前置条件

- Flutter SDK >= 3.22.0
- Python >= 3.10
- 阿里云 DashScope API Key（云端模式）
- Docker & Docker Compose（可选）

### 3.2 本地开发部署

```bash
# 1. 启动后端
cd backend
cp .env.example .env
# 填写 .env 中的 DASHSCOPE_API_KEY
uvicorn main:app --reload --port 8000

# 2. 启动 Flutter（开发模式）
flutter run

# 3. 配置 Flutter 连接地址
flutter run --dart-define=backend.url=http://localhost:8000
```

### 3.3 Docker 部署

```bash
cd /path/to/AIVideo
cp backend/.env.example backend/.env
# 填写 .env 中的真实凭证

docker-compose up -d
```

### 3.4 生产环境 Nginx

参考 `docs/nginx_sse_config.md` 配置反向代理。

## 五、已知限制

1. **端侧 VL 模型**：Qwen2-VL-2B GGUF 量化模型约 1GB，加载需 30-60 秒
2. **离线 ASR**：Whisper TFLite 需模型文件 `whisper-tiny-int8.tflite`（约 31MB）
3. **iOS 后台音频**：需要配置开发者证书和 App Group（用于实际部署）
4. **iOS Bundle Identifier**：需在 Xcode 中修改为唯一值（当前为 `com.example.aiVideo`）
5. **Android 前台通知**：Android 13+ 需要用户手动授权通知权限

## 六、后续迭代方向

5. **国际化**：支持英文、日文等语言
6. **Apple Watch 配件**：Watch 端快速提问，手机端执行推理
