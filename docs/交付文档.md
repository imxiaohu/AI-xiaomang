## 二、部署指南

### 2.1 后端部署（生产）

```bash
# 1. 服务器环境
apt update && apt install -y python3.10 python3.10-venv nginx certbot

# 2. 部署后端
cd backend
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# 3. 配置凭证
cp .env.example .env
nano .env  # 填入真实阿里云凭证

# 4. 配置 Nginx（参考 docs/nginx_sse_config.md）
sudo cp nginx.conf /etc/nginx/nginx.conf
sudo nginx -t
sudo systemctl reload nginx

# 5. 配置 HTTPS
sudo certbot --nginx -d your-domain.com

# 6. systemd 服务
sudo tee /etc/systemd/system/ai-video-backend.service <<EOF
[Unit]
Description=AI小芒 FastAPI Backend
After=network.target

[Service]
WorkingDirectory=/path/to/AIVideo/backend
ExecStart=/path/to/AIVideo/backend/.venv/bin/uvicorn main:app --host 127.0.0.1 --port 8000
Restart=always

[Install]
WantedBy=multi-user.target
EOF

sudo systemctl daemon-reload
sudo systemctl enable ai-video-backend
sudo systemctl start ai-video-backend
```

### 2.2 客户端配置

修改 `lib/providers/app_state.dart` 中的后端地址：

```dart
// 第243行附近
_sseService = SseStreamService(
  baseUrl: 'https://your-domain.com',  // 改为生产域名
  sessionId: _sessionId,
  token: 'your_token',  // 改为真实token
);
```

## 四、成本估算

### 4.1 阿里云费用（月估算，单用户）

| 服务 | 调用量 | 单价 | 月费用（元） |
|------|--------|------|------------|
| 实时 ASR | 30次/天 × 30天 × 30秒 | 0.18元/千次 | 4.86 |
| VL 推理 | 30次/天 × 30天 | 0.04元/次 | 36 |
| 流式 TTS | 30次/天 × 30天 × 50字 | 0.2元/万字 | 9 |
| **合计** | | | **~50元/月/用户** |

> 注：以上为标准定价参考，实际以阿里云官网为准。
> 百炼平台（新用户有免费额度）可大幅降低成本。

### 4.2 成本优化建议

1. **设置额度上限**：`.env` 中 `DAILY_TTS_QUOTA=10.0` 防止意外超支
2. **使用百炼API**：百炼平台性价比更高
3. **模型量化**：Whisper int8 + VL Q4_K_M 已为最小量化版本
4. **抽帧压缩**：`640x480 JPG65` 已是成本最优配置

## 五、更新日志

### v1.0.0（初始版本）

- SSE 流式通信
- 阿里云 ASR/VL/TTS 集成骨架

### v1.1.0（集成补全）

**后端修复**
- `Session` 类补全缺失字段（`audio_buffer`、`frame_buffer`、`transcribed_text`、`inference_task`）
- `/upload/chat/end` 端点改为触发实际 VL+TTS 推理流程
- TTS chunk 计数修复（`None` 返回时不再递增索引）
- SSE 推理逻辑抽取为共享函数 `trigger_session_inference`（被 SSE 事件流、`/chat/infer`、`/upload/chat/end` 三处复用）
- 后端 ASR（`AliyunASR`）集成到推理流程（客户端未做 ASR 时自动调用）
- 新增 `backend/Dockerfile`（多阶段构建）
- 新增 `backend/nginx.conf`（可直接部署，含 SSE 上行/下行全部配置）

**Flutter 修复**
- `AudioRecorderService` 接入 `record` 插件（16KHz PCM 真实录音）
- `TtsService` 播报完成后触发 `goIdle()` 回调
- SSE 客户端心跳超时重置修复（任何数据均重置计时器）
- SSE 解析支持多行 `data:` 字段（事件累积缓冲区）
- `endTurn()` URL 修复（`/chat/end` → `/upload/chat/end`）
- `baseUrl` 提取为 `lib/config/env_config.dart`（支持 `--dart-define` 注入生产地址）

**ProGuard**
- 补充完整规则（`record`、`audioplayers`、`flutter_tts`、阿里云 SDK）
